# Layout Modes

DocuShark adapts its workspace to what you're doing. Whether you're heads-down
writing, sketching a diagram, or working both at once, a **layout mode** arranges
the document panel, canvas, and properties to suit the task — so you're not
fighting the UI to get the room you need.

## The Four Layouts

| Layout | Best for | At a glance |
|--------|----------|-------------|
| **Relaxed** | Writing | Writing-first. Canvas one tap away. |
| **Designer** | Diagramming | Diagram-first. Document one tap away. |
| **Technician** | Doing both | Balanced split. Panels fly out on demand. |
| **Power** | Dense work | Everything visible. Tightest density. |

- **Relaxed** puts your document front and center as a comfortable reading column,
  with the canvas a tap away when you need it. It's the default.
- **Designer** flips that: the canvas dominates, and the document panel tucks away
  until you call it.
- **Technician** is a balanced split of prose and canvas, with the properties panel
  flying out only when you need it.
- **Power** keeps everything on screen at once, at the tightest density — for when
  you want every control in reach.

## Switching Layouts

Change layout any time — there are three ways:

- The **layout selector** chip in the toolbar (with a preview of each option)
- The keyboard: <kbd>Cmd/Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>1</kbd>–<kbd>4</kbd>
- The command palette (<kbd>Cmd/Ctrl</kbd> + <kbd>K</kbd>) → "Switch to … layout"

Your layout is an **app-level** setting — a single active mode for the whole
editor, not a per-document one. Customizations you make to one layout (say, moving
the properties panel) stay scoped to that layout and don't disturb the others.

## Focus in Relaxed

Because Relaxed is writing-first, it adds a **Write · Split · Diagram** focus
control so you can shift emphasis without leaving the layout:

- **Write** — just the document, distraction-free
- **Split** — document and canvas side by side (collapses to a single pane when the
  window is narrow)
- **Diagram** — canvas primary, prose tucked away

Switch focus from the control in the toolbar or with
<kbd>Cmd/Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>\\</kbd>.

## Custom Window Chrome

If you prefer a more app-like window, you can opt into DocuShark's own title bar
instead of the operating system's, from **Settings → Layout**. It's optional and
off by default; turning it on takes effect after a reload.
